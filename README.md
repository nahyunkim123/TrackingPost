## 상세설명
 - 내용 : Next.js, tailwindcss, SMART 택배 API를 활용한 실시간 택배 배송조회 웹사이트입니다
<br>
 - 구현 기간 :  2023.08.20 ~ 2023.08.23

## 구현 기능

- 객체 데이터를 사용한 색상 테마 변경 기능
- Afilter 함수를 사용하여  API 데이터 내 국내외 택배회사 옵션 분류
- smart 택배 API를 활용한 국내외 택배 배송 조회 기능
- 반응형 구현(모바일 / PC)
